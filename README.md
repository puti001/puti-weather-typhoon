# 🌀 Typhoon Tracker — OpenCode Skill

追蹤颱風動態，根據你的居住地提供在地化衝擊評估與天氣預報。

## 特色

- **只查 2 個中央氣象署網頁**，快速精準
- **第一次對話自動問你住哪裡**，設定一次就好
- **智慧衝擊分級**：知道颱風離你家多遠、在哪個方向
- **在地天氣**：目前氣溫、降雨機率、未來三天預報
- **人話回報**：不用看經緯度，直接告訴你嚴不嚴重

## 安裝

1. 下載 `typhoon-tracker/` 資料夾到你的 OpenCode skills 路徑
2. 在 `opencode.json` 加入技能路徑
3. 重啟 opencode
4. 跟 AI 說第一句話，它會問你住哪裡

## 依賴

- OpenCode (支援 Playwright 瀏覽器工具)
- 中央氣象署開放資料
